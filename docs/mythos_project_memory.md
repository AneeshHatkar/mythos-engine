## Global MythOS Rule

Every generated thing must be uniquely named, deeply explained, lore-connected, story-useful, character-affecting, plot-relevant, memory-aware, validated, provenance-tracked, anti-generic, and not one-line filler.

This applies to people, characters, families, countries, political units, cities, villages, regions, species, flora, fauna, creatures, objects, artifacts, roads, rivers, mountains, forests, seas, religions, gods, myths, rituals, laws, languages, factions, foods, clothing, weapons, tools, schools, markets, wars, historical eras, secret places, legends, and everything else MythOS generates.
